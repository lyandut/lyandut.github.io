# 基础快捷键

快捷键 功能
space 快速浏览
⌘+C 复制
⌘+X 剪切
⌘+V 粘贴
⌘+W 关闭
⌘+Q 退出
⌘+option+Esc 强制退出
⌘+Z 撤销
⌘+A 全选
⌘+S 保存
⌘+delete 删除／卸载
⌘+space spotlight搜索栏
control+space／caps lock 切换中英文输入法
⌘+tab 切换app

目录 功能
